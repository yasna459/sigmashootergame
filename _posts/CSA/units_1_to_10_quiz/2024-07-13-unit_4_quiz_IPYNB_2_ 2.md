## Unit 4: Iteration
What is the output of the following code?
```java
for (int i = 0; i < 5; i++) {
    System.out.print(i + " ");
}
```
a) 0 1 2 3 4<br>
b) 1 2 3 4 5<br>
c) 0 1 2 3 4 5<br>
d) 1 2 3 4

Answer: a) 0 1 2 3 4

How many times will the loop execute?
```java
int count = 0;
while (count < 3) {
    count++;
}
```
a) 2<br>
b) 3<br>
c) 4<br>
d) 1

Answer: b) 3

Which type of loop is guaranteed to execute at least once?<br>
a) for<br>
b) while<br>
c) do-while<br>
d) none of the above

Answer: c) do-while
