```python
def fibonacc(n):
    if n <= 0:
        return "invalid"
    elif n == 1:
        return 0
    elif n == 2:
        return 1
    else:
        return fibonacc(n - 1) + fibonacc(n - 2)


n = 10 
print(f"The {n}th Fibonacci number is: {fibonacc(n)}")
```

    The 10th Fibonacci number is: 34
